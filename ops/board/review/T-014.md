## Why
Three role files each get their slice of the hands-free contract. The planner learns to read the
human's standing ops/ROADMAP.md when no objective was given and to tag a conductor run's tasks
with a plan id so the backlog drain can tell "this plan" mechanically. EVOLVE gets the
auto-reversible apply mode locked to a fixed inert allowlist, with self-escalation explicitly
forbidden. The builder gets default-safe: defaulting only reversible, low-stakes spec details,
each recorded as an assumption — seam ambiguity and risk:high keep today's hard paths.
